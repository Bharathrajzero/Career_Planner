# Career Planner

Career Planner is a smart application designed to guide students and early‑career professionals in building personalized career roadmaps. It leverages agentic AI workflows to analyze user skills, interests, and goals, then generates tailored recommendations for learning paths, projects, and job readiness.

## 🚀 Features
- Personalized career path recommendations
- Skill tracking and visualization
- Project suggestions aligned with industry needs
- Lightweight data storage with SQLite
- Built with React (frontend) and Node.js/Express (backend)

## 🛠️ Tech Stack
- **Frontend:** React, JavaScript
- **Backend:** Node.js, Express
- **Database:** SQLite
- **AI Workflows:** Agentic AI integration
